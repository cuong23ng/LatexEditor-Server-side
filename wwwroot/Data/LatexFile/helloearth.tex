\begin{displaymath}
\frac{\Delta y}{\Delta x} = \frac{y_{1} - y_{0}}{x_{1} - x_{0}}
\end{displaymath}
Given a function f(x), the derivative of f with respect to x can be found from first
principles using:
\begin{displaymath}
\frac{df}{dx} = \lim_{\Delta x\rightarrow 0} = \frac{f(x+\Delta x) - f(x)}{\Delta x}
\end{displaymath}
